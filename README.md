# DariLik – AI-powered Rental Matching Platform

## Overview
DariLik is an AI-powered platform designed to improve the rental process in Morocco by helping landlords evaluate and match tenants efficiently.

## Problem
The rental market lacks transparency and structured evaluation, making tenant selection inefficient and risky for landlords.

## Solution
We developed a system that analyzes tenant profiles and provides intelligent scoring and matching based on financial stability, reliability, and compatibility.
